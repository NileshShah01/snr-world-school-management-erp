# Classe365

## Overview
Classe365 is the closest comparable business model to SNR WORLD — a multi-tenant SaaS platform serving schools of all sizes with a unified SIS + LMS + CRM + Finance platform. Operating globally with 10K+ schools, it represents the benchmark SNR is aiming to match and surpass.

## Feature List
- Student Information System (SIS)
- Learning Management System (LMS)
- CRM for admissions & enquiries
- Finance & accounting (invoicing, payment gateway)
- Attendance, timetable, exams
- Gradebook & report cards
- Multi-campus support
- Open API for integrations
- Mobile apps (iOS/Android)

## Pricing
- **₹15–50/student/year** (approximately ₹1.25–4.17/student/month), varying by modules selected
- Premium pricing for enterprise features

## Strengths
- Unified platform — SIS + LMS + CRM + Finance in one product (no add-on silos)
- Open API enables custom integrations
- CRM module for admissions pipeline management
- Built-in LMS — a critical differentiator
- Good mobile apps for parents, teachers, and students
- Multi-campus and multi-tenant by design
- 10K+ school adoption validates the model

## Weaknesses
- UI becomes increasingly complex at scale (too many menus)
- API documentation is decent but not world-class
- Premium pricing can be prohibitive for small schools
- No bundled public website CMS
- No ID card template engine

## What SNR Can Learn
1. **Unified platform strategy** — Classe365's SIS+LMS+CRM+Finance model validates all-in-one as the winning approach
2. **Open API** — enables ecosystem growth and enterprise sales
3. **Mobile-first** — their app investment is what makes them sticky
4. **CRM for admissions** — solving the enrolment funnel is a high-value feature
5. **Multi-tenant architecture** — same model as SNR, proving scalability
