# Education Desk (educationdesk.in)

## Overview
Education Desk is Apex Public School's current vendor, serving 800+ schools and 1M+ students across India. It is an established school management system with a broad feature set covering core SIS, attendance, fees, timetable, exams, and report cards.

## Feature List
- Student Information System (SIS)
- Attendance management
- Fee management & payment gateway
- Timetable scheduling
- Exam management
- Report cards
- Parent communication via SMS
- Transport management
- ID card generation
- Mobile parent app
- GPS tracking

## Pricing
- **₹5/student/month** — highly competitive for schools seeking per-student pricing

## Strengths
- Proven at scale (800+ schools, 1M+ students)
- Working mobile app for parents
- Built-in SMS gateway for parent communication
- Integrated payment gateway (UPI support)
- GPS tracking for transport
- Already a trusted incumbent at Apex Public School

## Weaknesses
- No public website CMS at all
- No multi-tenant / super admin support
- Basic ID card generation (templates minimal)
- No AI features (question formatting, analytics)
- No question paper formatter
- No library management module
- No LMS or digital content
- No WhatsApp integration

## What SNR Can Learn
1. **Working mobile app** — Education Desk has a functional parent app; SNR must prioritize app development
2. **SMS/WhatsApp integration** — seamless parent communication is a must-have
3. **Payment gateway** — in-app fee collection is expected by schools
4. **GPS tracking** — transport GPS is a table-stakes feature for safety-conscious schools
